# whs_update 版本日志模板

每个版本在 whs_update 文件顶部插入，格式如下（分类标题按实际内容选填）：

```
vX.Y (YYYY-MM-DD) — 版本名称
------------------------------------------
🎯 新功能
  - 功能 1：描述
  - 功能 2：描述

⚖️ 算法升级
  - 算法变更说明

🔧 代码改进
  - 架构/重构说明
  - 常量/配置变更

🐛 修复
  - P0-1: 严重问题修复
  - 一般 bug 修复

🧪 测试
  - tests/xxx.py 新增 N 个测试用例（共 M 个）
  - 覆盖说明


vX.Y-1 (上一版本日期) — 上一版本名称
------------------------------------------
...
```

## 规则

1. 日期使用 `YYYY-MM-DD` 格式
2. 分类标题只写有内容的分类（如没有 🧪 测试改动就不写）
3. 描述用中文，简洁一句
4. vX.Y 与 vX.Y-1 之间空一行
5. 注释中的 P0/P1/P2/P3 优先级标注保留原样
